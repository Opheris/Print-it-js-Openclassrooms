# Print-it-js-Openclassrooms

## Projet OpenClassrooms - Initiation à JavaScript

Ce projet est une introduction au JavaScript dans le cadre d'un parcours OpenClassrooms. L'objectif est de créer un carrousel (slider) dynamique pour le site web de l'imprimerie Print It.

### Objectifs du projet

1. Ajouter des écouteurs d'événements aux flèches de navigation
2. Créer des points de navigation (bullet points) pour le slider
3. Modifier dynamiquement l'image et le texte affichés lors du clic sur les flèches
4. Implémenter un défilement infini du carrousel

### Structure du projet

```
Print-it-js-Openclassrooms/
│
├── assets/
│   ├── images/           # Images du site (logo, icônes, etc.)
│   │   └── slideshow/    # Images du carrousel
│   ├── style.css         # Styles CSS du site
│   └── script.js         # Script JavaScript (à compléter)
├── index.html            # Page HTML principale
└── README.md
```

### Utilisation

1. Ouvrir le fichier `index.html` dans un navigateur web
2. Les modifications JavaScript doivent être effectuées dans `assets/script.js`

### Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
